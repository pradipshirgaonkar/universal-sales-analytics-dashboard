# Universal Sales Analytics Dashboard for Consumer Durables

A Python-based dashboard for sales analysis in Consumer Durables sector. Features include performance tracking, campaign evaluation, customer segmentation, and retention insights.

## Setup
1. Install Python 3.11 and PostgreSQL.
2. `pip install -r requirements.txt`
3. Run `python data_loader.py` to populate DB.
4. `streamlit run app.py`

## Usage
- Filter data in sidebar.
- View KPIs, charts, and tables.

## Tech
- Python 3.11, Streamlit, Plotly, Pandas, SQLAlchemy, PostgreSQL
